# Log

本文档记录已发生的重要事件、决策、变更、事故、验收和复盘，是项目治理历史账本，也是重要决策记忆。重要决策应记录原因、影响和结果；若它形成长期约束，应按记忆路由同步到对应正式治理文件。

当用户明确裁决某方向拒绝、不做、暂不支持或不再重复建议时，应记录原因和取舍，避免未来 agent 反复重开同一问题。本文只记录决策事件和依据，不复制已有 artifact 全文。

## 记录格式

```text
时间：
类型：初始化 / 变更 / 决策 / 事故 / 验收 / 复盘
摘要：
原因：
影响范围：
结果：
后续：
```
